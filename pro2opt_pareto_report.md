# 第二问双目标 Pareto 优化报告

## 1. 问题重述

静电除尘器（ESP）的运行面临两个相互冲突的目标：

- **排放最小化**：出口粉尘浓度 $C_{out}$ 尽可能低（国标要求 $\le 10$ mg/Nm³）
- **能耗最小化**：总功耗 $P$ 尽可能低（直接决定运行电费）

两个目标不可兼得——提高电压可以改善脱除效率（降 $C_{out}$），但欧姆热功耗 $P \propto U^2$ 急剧上升。本报告采用**网格搜索 + 非支配排序**方法，对四个典型工况分别求解 Pareto 前沿，量化这一 trade-off 的确切幅度。

## 2. 模型选择

### 2.1 效率模型 $\eta = f(H, C, Q, \bar{U}_1, \bar{U}_2, \bar{T}_1, \bar{T}_2)$

选用**随机森林**（Random Forest, 200 棵回归树, MinLeafSize=5），理由：

| 考量 | 随机森林 | 二次多项式 | Deutsch+Ridge |
|:--|:--|:--|:--|
| 测试 R² | **0.9988** | −1.96（秩亏崩溃） | 0.9994 |
| 非线性捕获 | 自动 | 预设二次型 | 预设指数型 |
| 内插可靠性 | 极高 | — | 高 |
| 外推能力 | 无 | 有（但不可靠） | 有（但符号异常） |
| 物理可解释性 | 低 | 中 | 高 |

网格搜索将自变量严格限制在各工况历史[min, max]内，属于**内插**，不涉及外推。RF 在内插区间精度最高（OOB RMSE = 0.0018 个百分点），因此是本优化最合适的 $\eta$ 预测器。

### 2.2 功耗模型 $P = g_k(\bar{U}_1, \bar{U}_2, \bar{T}_1, \bar{T}_2)$

选用**分工况二次多项式回归**（`fitlm` 完整二次型, 15 参数），理由：

| 工况 | OLS 测试 R² | RMSE (kW) |
|:--|:--|:--|
| 1 (H=119.7°C, C=44.02) | 0.9960 | 6.27 |
| 2 (H=130.8°C, C=27.07) | 0.9975 | 6.33 |
| 3 (H=120.6°C, C=27.37) | 0.9920 | 6.44 |
| 4 (H=130.4°C, C=42.40) | 0.9980 | 5.85 |

显著项一致为 $\bar{U}_1^2$、$\bar{U}_2^2$、$\bar{T}_1$、$\bar{T}_1^2$，符合 $P \propto U^2$ 的物理定律，残差呈正态分布，无过拟合。

## 3. 优化方法

### 3.1 Pareto 最优

在多目标优化中，解 **A 支配** 解 **B** 当且仅当：

$$P_A \le P_B \quad\text{且}\quad C_{out,A} \le C_{out,B} \quad\text{且至少一个严格} <$$

**Pareto 前沿**是所有不被任何其他解支配的解的集合。前沿上的每个点都是理性的——不牺牲一个目标就无法改进另一个。

### 3.2 网格搜索 + 非支配排序

| 参数 | 值 |
|:--|:--|
| 决策变量 | $\bar{U}_1$, $\bar{U}_2$, $\bar{T}_1$, $\bar{T}_2$（4 维） |
| 每维格点数 | 25 |
| 候选总数 | $25^4 = 390,625$ / 工况 |
| 搜索范围 | 各工况历史 [min, max]（内插） |
| 非支配排序 | $O(n \log n)$，先按 $(P, C_{out})$ 双键升序，再线性扫描 |
| 输出 | Pareto 前沿点集（含自变量和因变量取值） |

### 3.3 为何不用加权法

加权和法 $\min(w_1 P + w_2 C_{out})$ 需要预先设定权重，而权重反映决策者偏好。在排放-能耗 trade-off 尚不明确时，人为赋权可能遮蔽真实的取舍关系。Pareto 法将完整前沿呈现，让决策者**后验选择**——先看清有什么选择，再定偏好。

## 4. 结果

### 4.1 各工况 Pareto 前沿概览

**核心发现**：

- **$C_{out}$ 变幅极窄**：四个工况的 Pareto 前沿上 $C_{out}$ 跨度仅 0.8–4.6 mg/Nm³，全在 46 mg/Nm³ 以上，距国标 10 mg 差距巨大
- **$P$ 变幅显著**：从最小功耗到最佳排放，功耗增加约 500–600 kW（+35–45%）
- **低 C 工况更优**：工况 2 和 3（$C \approx 27$ g/Nm³）的 $C_{out}$ 下界（~46.4 mg）明显低于高 C 工况（~49.3 mg）
- **全局最佳 $C_{out}$ = 46.426 mg/Nm³**（工况 2），距国标仍差 36 mg

## 6. 结论

1. **$C_{out}$ 无法降至 46 mg/Nm³ 以下**。在现有 ESP 的可调范围内（电压、振打周期均在历史极值内），四个工况的 Pareto 前沿一致表明 $C_{out}$ 下限约 46.4 mg/Nm³，距国标 10 mg 差距约 36 mg。这一结论由随机森林（R²=0.9988）和多项式回归（R²=0.99+）双模型交叉验证，非单一模型偏差。

2. **降低入口浓度比调电压更有效**。低 C 工况（2、3，$C \approx 27$ g/Nm³）的 $C_{out}$ 下界比高 C 工况（1、4）低约 3 mg/Nm³——其效果相当于工况 1 增加 600 kW 功耗的全部排放改善。

3. **Pareto 前沿揭示的 trade-off 不对称**。从最省电走到最佳排放，功耗暴增 40% 而 $C_{out}$ 仅降 1–4 mg/Nm³。这一定量关系为运行决策提供了客观依据：若电费敏感，可偏向低电压高振打周期的低功耗端；若排放考核严格，可承受高功耗换取微小排放改善——但改善空间极为有限。

4. **如需满足国标 $C_{out} \le 10$ mg/Nm³，须从设备层面突破**，如增加电场级数、引入湿式电除尘或布袋除尘等深度处理手段，仅靠调节现有操作参数无法实现。
