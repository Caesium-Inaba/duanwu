# 第二问 f 模型：随机森林回归 η = f(H, C, Q, Ū₁, Ū₂, T̄₁, T̄₂)

## 1. 为什么用随机森林

静电除尘的脱除效率 η 受七个操作变量共同影响，变量之间存在非线性耦合（如 Deutsch 公式中电压以指数形式进入，温度影响迁移率），经典线性回归难以捕捉此类复杂关系。随机森林（Random Forest, RF）是一种基于决策树集成的非参数回归方法，具备以下优势：

- **自动建模非线性与交互**：无需预设函数形式（如二次型或 Deutsch 指数型），树的分裂天然捕获高阶交互
- **对共线性不敏感**：Bagging 和随机子空间使得即使预测变量间高度相关，模型仍能稳定收敛
- **提供变量重要性排序**：基于 OOB（Out-of-Bag）置换的预测误差增量，量化每个变量对 η 的相对贡献
- **OOB 误差自带交叉验证**：每棵树约 1/3 的样本未参与训练，聚合这些"包外"预测可无偏估计泛化误差

## 2. 模型设定

| 项目 | 设定 |
|:--|:--|
| 算法 | Bagging 集成（`TreeBagger`） |
| 基学习器 | 回归树（CART），`MinLeafSize = 5` |
| 树数量 | `nTrees = 200` |
| 预测变量（7 个） | $H$（温度, °C）、$C$（入口浓度, g/Nm³）、$Q$（烟气量, Nm³/h）、$\bar{U}_1$（一二电场均值, kV）、$\bar{U}_2$（三四电场均值, kV）、$\bar{T}_1$（一二电场振打周期, s）、$\bar{T}_2$（三四电场振打周期, s） |
| 响应变量 | $\eta$（脱除效率, %） |
| 训练集 | 7366 行（80%） |
| 测试集 | 1842 行（20%） |
| 随机种子 | `rng(42)` 确保可复现 |
| 有效样本 | 9208 行（剔除 NaN 后） |

**模型记为**：$\eta = \hat{f}_{\text{RF}}(H, C, Q, \bar{U}_1, \bar{U}_2, \bar{T}_1, \bar{T}_2)$

输出文件：`rf_models.mat` → `f_model`（TreeBagger 对象，约 49 MB）

## 3. 拟合效果

### 3.1 核心指标

| 指标 | 数值 | 说明 |
|:--|:--|:--|
| OOB R² | **0.9981** | 包外数据决定系数，接近 1，拟合极好 |
| OOB RMSE | **0.0018 百分点** | η 的预测误差仅约 0.002%，几乎无误差 |
| 训练 R² | 0.9992 | 训练集上几乎完美 |
| 测试 R² | **0.9988** | 独立测试集，未见衰退 |
| ΔR²（训练−测试） | 0.0004 | 远小于过拟合警戒线 0.02 |

### 3.2 过拟合诊断

```
训练 R² − 测试 R² = 0.0004 ≪ 0.02 ✅ 无过拟合
OOB R² = 0.9981 ≈ 测试 R² = 0.9988 ✅ 泛化能力一致
```

200 棵树 + MinLeafSize=5 的组合在保证拟合精度的同时约束了叶子节点最小样本数，防止单棵树记忆噪声。训练集与测试集的 R² 几乎无差距，说明模型学到的规律是数据内在结构而非噪声。

### 3.3 什么是 R² = 0.9988 的水平

在 η ≈ 95%–99.9% 的区间内，预测误差（RMSE）仅约 0.002 个百分点。直观地说：**给 RF 七个操作参数，它对除尘效率的预测与实测值在小数点后第三位才出现偏离**。

## 4. 变量重要性

通过 OOB 置换重要性（Predictor Importance）衡量：将某变量在包外样本中随机打乱，观察 OOB 均方误差的增大量。增量越大，该变量对 η 的贡献越大。

| 排序 | 变量 | 重要性 | 相对贡献 |
|:--|:--|:--|:--|
| 1 | $C$（入口浓度） | **7.7709** | 基准（100%） |
| 2 | $Q$（烟气量） | 0.7736 | 10.0% |
| 3 | $H$（温度） | 0.7591 | 9.8% |
| 4 | $\bar{T}_2$（三四电场振打周期） | 0.7177 | 9.2% |
| 5 | $\bar{T}_1$（一二电场振打周期） | 0.7153 | 9.2% |
| 6 | $\bar{U}_1$（一二电场电压） | 0.5611 | 7.2% |
| 7 | $\bar{U}_2$（三四电场电压） | 0.4647 | 6.0% |

### 关键发现

**C（入口浓度）的重要性是第二名 Q 的 10 倍，呈现绝对主导。** 这意味着随机森林从数据中学到的核心规律是：η 的变化几乎完全由 C 决定，其余六个变量对 η 的贡献加在一起不足 C 的 50%。

这一现象并非 RF 的缺陷。事后诊断（`c_eta.m`）证实：

- $C$ 与 $\eta$ 的 Spearman 秩相关系数 **ρ = 0.9999**
- 出口浓度 $C_{out}$ 实测范围 $[48.74, 50.00]$ mg/Nm³，跨度仅 1.26 mg/Nm³，几乎恒定

由于 $C_{out} \approx 49.5$ 常数，η 由定义式 $\eta = 1 - C_{out}/(C \times 1000)$ 几乎完全被 $C$ 锁定。电压和振打周期的调节对 η 的影响被淹没在 $C_{out}$ 的测量波动中，RF 正确地捕捉到了这一物理现实。

## 5. 模型适用范围

RF 是非参数插值器——它在训练数据的凸包内部预测精度极高（R² > 0.99），但**不能外推**到训练数据未覆盖的变量组合。四个工况的变量范围如下：

| 工况 | H (°C) | C (g/Nm³) | Ū₁ (kV) | Ū₂ (kV) | T̄₁ (s) | T̄₂ (s) |
|:--|:--|:--|:--|:--|:--|:--|
| 1 | [112, 125] | [35, 54] | 聚类相关 | 聚类相关 | 聚类相关 | 聚类相关 |
| 2 | [126, 138] | [18, 35] | 聚类相关 | 聚类相关 | 聚类相关 | 聚类相关 |
| 3 | [113, 126] | [18, 36] | 聚类相关 | 聚类相关 | 聚类相关 | 聚类相关 |
| 4 | [125, 138] | [35, 53] | 聚类相关 | 聚类相关 | 聚类相关 | 聚类相关 |

在此范围内，RF 预测可靠；范围外的预测（如 $C_{out} \le 10$ mg/Nm³ 对应的 η ≥ 99.98%）不可信，这也是网格搜索（`pro2opt.m`）不可行点为零的根本原因。

## 8. 结论

随机森林以 **R² = 0.9988、RMSE = 0.002 个百分点**的精度拟合了 η 与七个操作变量之间的映射关系。变量重要性分析揭示**入口浓度 C 是 η 的主导变量**——这一发现并非模型缺陷，而是由 $C_{out}$ 近乎恒定的物理现实所决定的。RF 模型在数据范围内的预测精度足以支撑后续 Pareto 优化（`pro2opt_lease.m`），但不能用于数据范围外的外推。
